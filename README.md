## Episode 1 — GSC Bulk Data Pull

Pull 51,000+ keywords from Google Search Console in 30 seconds.

GSC shows 1,000 rows. This script pulls everything.

### What you need:
- Python installed (python.org)
- Google Cloud credentials.json
- Run: py -m pip install google-auth-oauthlib google-auth-httplib2 google-api-python-client pandas openpyxl

### How to run:
1. Add your credentials.json to the folder
2. Open the script and change SITE_URL to your domain
3. Run: py gsc_pull.py
